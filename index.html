Welcome to µSploit Framework home page

µSploit framework is very basic exploit/peneration test tool framework. Inspirited by websploit and metasploit. Websploits development has been sadly ended years ago so i decided to write my own. I have been focused to make µSploit easy (user friendly) and simple to use as possible. µSploit is written in python 3, and developed on ubuntu.
