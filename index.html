µSploit Framework status:

This is official page of µSploit Framework project.

µSploit is basic penetration testing tool.

Basic features:

  • Run exploits
  • Simple UI
  • Includes scapy
  • Easy to use
  • Easy to write your own exploit (module)
  • Scripting

Available modules:

  • apache_users
  • arp_dos
  • arp_spoof
  • bluetooth_pod
  • cloudflare_resolver
  • dir_scanner
  • dns_spoof
  • email_bomber
  • hostname_resolver
  • mac_spoof
  • mitm
  • network_kill
  • pma_scanner
  • port_scanner
  • web_killer
  • wifi_jammer
